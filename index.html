Какие проблемы?

Различия платформ

Отличаются:
  • Файловая структура приложения
  • API встроенных оъектов
  • Коды клавиш

Решение:
Разделять на этапе сборки

Низкое быстродействие

  • Чем меньше нод — тем лучше
  • div и span — еще лучше
  • Использовать короткие селекторы, селекторы по классам
  • Меньше CSS3
  • Использовать jQuery VanillaJS и полифиллы
